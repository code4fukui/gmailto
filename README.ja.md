# gmailto

gmailtoは、Gmailで簡単にメールを送れる小さなウェブアプリケーションです。

## デモ
[https://code4fukui.github.io/gmailto/](https://code4fukui.github.io/gmailto/)

## 機能
- 宛先、CC、BCC、件名、本文を入力してGmailのリンクを生成
- 生成されたリンクをクリックするとGmailが開き、入力内容が反映される

## 使い方
1. [デモページ](https://code4fukui.github.io/gmailto/)を開く
2. 必要な情報を入力フォームに入力する
3. 生成されたメールリンクをクリックすると、Gmailが開きます
